## LosP Finance — Sistema de Gestão Financeira Pessoal

LosP Finance é uma plataforma web desenvolvida para ajudar usuários a controlar seus gastos, definir metas financeiras, acompanhar evolução mês a mês e visualizar insights de maneira clara e intuitiva.
O projeto combina Front-end moderno com Back-end em Java, trazendo uma experiência completa, organizada e profissional.

## Objetivo do Projeto

- Criar um sistema financeiro simples de usar, mas com recursos avançados de análise, que permita ao usuário:
- Registrar e categorizar seus gastos e rendas
- Definir metas financeiras mensais e anuais
- Acompanhar o progresso por meio de gráficos e relatórios
- Comparar meses diferentes para entender evolução
- Personalizar categorias para sua própria realidade
- O foco é oferecer controle, clareza e visualização, tudo dentro de uma interface limpa em conjunto com um back-end robusto em Java.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Java
- MySQL

## Funcionalidades

- Registro de transações: Gastos, rendas, categorias personalizadas, data e descrição.
- Painel financeiro: Total gasto, total recebido, saldo e categorias mais caras.
- Metas financeiras: Metas mensais/anuais com barra de progresso e alertas.
- Comparação mensal: Evolução mês a mês, comparação por categorias e ranking dos meses mais caros.
- Gráficos: Visualizações claras dos gastos e tendências.
